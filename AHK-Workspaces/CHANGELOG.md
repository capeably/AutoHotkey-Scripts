# Changelog

All notable changes to AHK-Workspaces will be documented in this file.

### Changed

- Icon path now uses `A_ScriptDir` for reliable resolution regardless of launch method

### Added

## [1.0.0] - 2026-04-10

### Added
- Initial release with workspace save/restore functionality
